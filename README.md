### 🧭 About me

I'm a **Data Systems & BI Developer** with **15+ years** of experience turning complex, disconnected data into reliable, governed, and actionable information. I design and build the platforms — data warehouses, ETL pipelines, dashboards, and web applications — that help healthcare, nonprofit, and public-sector teams make decisions with confidence.

- 🔭 I build **BI platforms and data systems** end-to-end: source data → warehouse → model → dashboard → app
- 🏥 Deep focus on **healthcare & public-sector data** (HMIS, DHIS2, indicator design, health expenditure)
- 🧱 I develop **ASP.NET Core** applications and architect **SQL Server** warehouses with SSIS/ETL
- 📊 I turn reporting into insight with **Power BI &amp; Microsoft Fabric**
- 🤝 I've delivered for organizations including **UNICEF, MSH, MOPH, MSI, and Palladium**
- 💬 Ask me about **Power BI, Microsoft Fabric, SQL Server warehousing, ASP.NET Core, and report modernization**
